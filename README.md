# TechWalk Website

The **TechWalk Website** is the official platform for TechWalk. It provides information about the organization, highlights community initiatives, and serves as a central hub for students, professionals, and technology enthusiasts interested in TechWalk events and activities.

The site is built using modern web technologies with a focus on performance, scalability, and clean component-based design.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- pnpm / npm

## Getting Started

Clone the repository:

```
git clone https://github.com/lisha080/TechWalk_Website.git
cd TechWalk_Website
```

Install dependencies:

```
pnpm install
```

or

```
npm install
```

Run the development server:

```
pnpm dev
```

or

```
npm run dev
```

Open the project in your browser:

```
http://localhost:3000
```
